Приложение написано на Java с использованием Spring Boot. Для хранения данных используется PostgreSQL, а миграции управляются Flyway.
Можно получать курс валюты, конвертировать суммы между валютами и просматривать историю всех конверсий.
Чтобы запустить сервис, нужно настроить подключение к базе данных в application.properties и запустить приложение.

API имеет три основных эндпоинта:
Получение курса валюты
Конвертация суммы из одной валюты в другую
Просмотр истории конверсий
